# todoMicroService

a **TODO LIST** application  written in Golang.

## Get Started:

1 - build app:

> **make build**

2 - run container:

> **make run**

3 - migrate tables:

> **go install -tags 'postgres' github.com/golang-migrate/migrate/v4/cmd/migrate@latest**
> **make migrate**

## Help Command:

> **make help**
